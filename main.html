How it Works?

  • Here the loan amount is calculated based on 'Type of Customer'
  • Remortgage - is you take another loan (2nd loan) on the same property.
  • Repayment - is paying back the entire loan
  • Interest Only - is a loan in which the borrower pays only the interest for some or all of the term, with the principal balance unchanged during the interest-only period.
  • In type of customer tab, we select the type based on the drop down list
  • Repayment type defines complete repayment or interest only
  • Property Value is the value you need to feed in manually
  • Deposit amount is the prepayment if any
  • Net Property Value will be calculated automatically, Property Value - Deposit Amount
  • Mortgage term you need to feed manually in number of years
  • Initial Interest Rate, is the rate you put manually
